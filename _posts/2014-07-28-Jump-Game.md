## 题意：
数组中每个元素代表能够跳跃的最大格数，从A[0]出发，问能否到达最后一点

## 要求：


## 思路：
遍历，每个点更新能够到达的最远距离，如果到达某点时最远距离已经小雨该点的距离，那么一定无法到达最远距离

## 代码：

### C++:

{% highlight c++ %}
class Solution {
public:
    bool canJump(int A[], int n) {
        int ans=0;
        for(int i=0;i<n;i++)
        {
            if(i>ans)
            {
                return false;
            }
            ans=max(ans,i+A[i]);
            if(ans>=n-1)
            {
                return true;
            }
        }
        return ans>=n-1;
    }
};


 {% endhighlight %}
### python:

{% highlight python %}
class Solution:
    # @param A, a list of integers
    # @return a boolean
    def canJump(self, A):
        ans=0
        for i in range(len(A)):
            if ans<i:
                return False
            ans=max(ans,i+A[i])
            if ans>=len(A)-1:
                return True
        return ans>=len(A)-1
